# Recipes

Типовые сценарии использования laravel-admin. Каждый рецепт — короткий
copy-paste пример с минимумом контекста.

| Рецепт | Описание |
|---|---|
| [custom-actions](custom-actions.md) | Bulk и row actions, modal-actions с формой, async-actions через delayed-process |
| [soft-deletes](soft-deletes.md) | Включить soft-delete + restore/forceDelete + TrashedFilter |
| [file-uploads](file-uploads.md) | FileUpload field + Wysiwyg image-upload + Image cropping |
| [multi-tenancy](multi-tenancy.md) | TenantResolver + TenantScoped trait для row-level фильтрации |
| [audit-trail](audit-trail.md) | Loggable trait + AuditTrail layout на view-screen |
| [plugin-development](plugin-development.md) | Структура AdminPlugin, регистрация Resource'ов из пакета |
| [import-export](import-export.md) | CSV/XLSX/PDF export + 4-step Import Wizard |
| [theming](theming.md) | Custom themes, переключатель, CSS-vars overrides |
